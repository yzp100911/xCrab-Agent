# 🦀 skillgate-agent

**skillgate-agent** — AI Personal Assistant Suite, featuring four core components: **xCrab (AI Execution Engine)**, **eclaw (Service Dispatcher)**, **cclaw (Remote Distributor)**, **wclaw (Web Client)**.

One repository download, complete deployment.
